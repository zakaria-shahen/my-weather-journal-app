# Weather-Journal App Project

## Overview

This project  to create an asynchronous web app that uses Web API and user data to dynamically update the UI.

## requirement

- install node.js
- install npm

## How to run

- install ->  run command

```shell
    npm install init
    npm install node
    npm install express
    npm install body-parser
```

- run command

```shell
    node server.js
```

## Added

1. simple Edit styles
2. Inquiry about weather conditions via API
3. using
    - body-parser@1.19.0
    - cors@2.8.5
    - express@4.17.1
    - node-fetch@2.6.1s
